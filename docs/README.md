# Documentation

## Project Overview

Analytics GPT is an AI-powered document assistant built using Retrieval Augmented Generation (RAG).

Users can upload PDF documents and ask natural language questions.

The application retrieves relevant document chunks and generates answers using Google's Gemini API.
